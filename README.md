# SkillForge

SkillForge is a personalized learning dashboard that provides AI-driven learning paths tailored to your goals and pace.

## Project Details
- **Project Name:** SkillForge
- **Author:** Mahdi
- **Description:** A modern web application for skill development and personal learning paths. This is a personal project created to help users track progress, get smart recommendations, and follow customized learning journeys.

## Tech Stack
- Frontend: React, Vite, TailwindCSS
- State Management: React Router

## Getting Started
To get started with SkillForge, make sure you have Node.js installed, then run the following commands:
```bash
cd frontend
npm install
npm run dev
```
